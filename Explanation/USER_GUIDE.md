# SETU User Guide

Welcome to SETU, the Enterprise Project Management System. This guide helps you navigate the platform and perform common tasks.

## 1. Getting Started

### 1.1 Login & Dashboard
- Access the system at the provided URL.
- Log in with your corporate credentials.
- The **Main Dashboard** gives you a high-level view of all your assigned projects.

### 1.2 Enterprise Project Structure (EPS)
- Navigate to **EPS** from the main menu.
- This is your starting point. It shows the hierarchy: Company -> Projects -> Towers.
- **Actions**:
    - Click **Settings** (Gear Icon) on a project to configure basic properties.
    - Click specific module buttons (e.g., **Plan**, **BOQ**, **Quality**) on a project card to enter that module.

## 2. Project Setup Workflow (For Managers)

1.  **Define Scope (BOQ)**:
    - Go to **BOQ Module**.
    - Import your Tender BOQ from Excel.
    - Use **Measurement Manager** to define the detailed quantities for each floor/unit.

2.  **Define Schedule (Plan)**:
    - Go to **Schedule Module**.
    - Create your WBS structure.
    - Add Activities and link them (Dependencies).
    - Map Activities to BOQ items (Cost Loading).

3.  **Setup Quality & Safety**:
    - Go to **Quality** -> **Checklists**: Configure inspection templates.
    - Go to **EHS**: Setup safety policy configurations.

## 3. Daily Site Workflow (For Site Engineers)

### 3.1 Morning: Labor & Safety
1.  **Labor Count**:
    - Go to **Labor Module**.
    - Log today's attendance (Subcontractor wise).
2.  **Toolbox Talk**:
    - Go to **EHS Module**.
    - Log the Daily Safety Induction/Briefing.

### 3.2 During Day: Execution & Quality
3.  **Inspections**:
    - When work is ready (e.g., Shuttering done), go to **Quality** -> **Inspections**.
    - Raise an Inspection Request (WIR).
4.  **Observations**:
    - If you see a safety issue, logging it in **EHS** -> **Observations**.
    - If you see a defect, log it in **Quality** -> **Snags**.

### 3.3 End of Day: Progress Reporting
5.  **Book Progress**:
    - Go to **Progress Module**.
    - Select your location (e.g., Block A, 1st Floor).
    - Enter quantity achieved for relevant items (e.g., "Concreting: 15 cum").
    - Submit DPR (Daily Progress Report).
